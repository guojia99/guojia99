### 9.1 go 构建so

编辑一个go函数

```go
package main

import "C"

//export Sum
func Sum(a, b int) int {
	return a + b + 1
}

func main() {}
```

执行如下命令：

```shell
go build -buildmode=c-shared -o xxx.so xxx.go
```

将会生成so文件和.h文件

**调用1**

```go
lib, err := dl.Open("/usr/local/lib/xxx.so", 0)
if err != nil {
    panic(err)
}
defer lib.Close()

var Sum func(a C.longlong, b C.longlong) int64
lib.Sym("Sum", &Sum)
data := Sum(C.longlong(a), C.longlong(b))
```

**调用2**



```go
package use

/*
#cgo CFLAGS: -IC./
#cgo LDFLAGS: -LC./ -lsum

	"fmt"
	"time"
)
